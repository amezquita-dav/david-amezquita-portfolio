# Industrial Logistics Monitoring Dashboard

## Overview
Industrial-style dashboard designed to visualize logistics operations and operational KPIs in real time.

This project focuses on building data-driven interfaces that help monitor trailer yard operations, carrier distribution, and operational performance.

## Problem
Logistics teams need better visibility into operational data to make faster decisions and track performance indicators.

## Solution
Develop a modular dashboard that integrates backend data with visual analytics components.

## Technologies
- Angular
- SQL
- REST APIs
- Data Visualization

## Key Features
- KPI visualization
- Dynamic filters
- Operational monitoring dashboards
- Modular UI components

## Architecture
Frontend dashboard connected to backend services that retrieve and process operational data from a relational database.
