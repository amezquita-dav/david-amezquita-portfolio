# Inventory Analytics (SQL + Python)

## Overview
Data analytics project focused on generating inventory insights using SQL and Python.

This system analyzes stock levels, inventory movements, and product performance to support operational decision-making.

## Problem
Organizations often lack visibility into inventory performance, leading to overstock, stockouts, and inefficient inventory management.

## Solution
Develop an analytics workflow that combines SQL queries with Python data analysis to generate operational insights.

## Technologies
- SQL
- Python
- Pandas
- Data Visualization

## Key Analytics
- Inventory turnover
- Slow-moving products
- Stock level analysis
- Inventory aging
- Reorder insights

## Architecture
Relational database used for structured inventory data, combined with Python notebooks for exploratory analysis and visualization.
